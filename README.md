# Консольный кликер (C++)

Консольная игра-кликер на языке C++.

## Описание

Игрок зарабатывает очки нажатием клавиши. Очки можно тратить на улучшения, которые увеличивают доход.

В игре реализован игровой цикл с обновлением состояния и автодоходом по времени.

## Возможности

1. Получение очков с помощью кликов.
2. Улучшение силы клика.
3. Автоматический доход (начисляется каждую секунду).
4. Отдельное меню улучшений.
5. Неблокирующий ввод (игра не "зависает" при ожидании команды).

## Управление

### Основное меню
1. c — клик (получить очки).
2. u — открыть меню улучшений.
3. q — выход.

### Меню улучшений
1. 1 — увеличить силу клика.
2. 2 — увеличить автодоход.
3. 0 — вернуться назад.

## Сборка и запуск

### Visual Studio 2022
1. Открыть папку проекта.
2. Добавить файл "main.cpp" в проект.
3. Собрать и запустить программу.
