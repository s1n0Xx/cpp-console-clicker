# Консольный кликер (C++)

Консольная игра-кликер на языке C++.

## Описание

Игрок зарабатывает очки нажатием клавиш. Можно покупать улучшения и увеличивать доход.

## Возможности

1. Клики для получения очков.
2. Улучшение силы клика.
3. Автоматический доход.
4. Консольный интерфейс.

## Управление

1. c — клик.
2. u — улучшение.
3. a — автодоход.
4. q — выход.

## Сборка и запуск

Visual Studio 2022/
